# Final-Year-Project
Deep Learning-Based Weather Forecasting: A Comparative Study of ARIMA, Random Forest, and LSTM Models.



PROPOSED WORK:
The aim of this project is to develop a weather forecasting system that compares different forecasting approaches using historical weather data.
The main objectives of the project are:
•	To collect and preprocess historical weather data from publicly available datasets.
•	To implement a traditional statistical forecasting model (ARIMA).
•	To implement a machine learning model (Random Forest Regression).
•	To develop a deep learning forecasting model using Long Short-Term Memory (LSTM).
•	To evaluate the performance of each model using forecasting accuracy metrics.
•	To compare the models and determine which approach provides the best prediction performance.
The project will involve data pre-processing, model training, model evaluation, and comparative analysis. Visualization techniques will also be used to compare predicted and actual weather values.
